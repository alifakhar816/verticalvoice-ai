## Recommendation

For defense: Lead with **engineering execution** (the system is built, deployed, and working) rather than **market projections** (unsourced). Use **verifiable sources** (HIPAA, Fair Housing Act, academic papers). Be **honest about assumptions**. Panels value integrity over inflated claims.
